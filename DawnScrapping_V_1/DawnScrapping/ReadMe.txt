"These instructions are for Windows OS only"
Instructions:
	1. If Visual Studio 2015 (V14), VC++, is not already installed, please install
	   VC++ build tools from the following link (http://download.microsoft.com/download/5/F/7/5F7ACAEB-8363-451F-9425-68A90F98B238/visualcppbuildtools_full.exe)
	2. This program is tested and recommended from Python3.6. If already installed move to step 3, if not than install it from the provided link
	   (https://www.python.org/ftp/python/3.6.4/python-3.6.4-amd64.exe)
	3. Check if python is already in your path. To check open cmd and type "python". If python is in path than install scrapy with following command
	   "python -m pip install scrapy". After that install scrapy-proxies by following this command "python -m pip install scrapy-proxies"
	4. Move to the base directory of the folder and run main file using following command "python main.py"
